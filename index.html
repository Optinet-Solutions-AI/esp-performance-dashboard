Choose the ESP this data belongs to. The file will be merged into that ESP's dataset and all related dashboards will update automatically.

Step 2 — Upload Activity File

Drop your XLSX or CSV export from the ESP. The file must contain at minimum: email, sent-time, delivery, opens, clicks, bounced, unsubscribed columns (Mailmodo format). Other ESP formats will be auto-detected.
